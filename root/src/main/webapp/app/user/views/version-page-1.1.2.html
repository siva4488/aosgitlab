Docker Configuration
Configure AOS locally with Docker (Ubuntu/Centos OS are supported) by running: git clone -b 1.1.2 https://github.com/aosapp/account-service.git AOS installation: cd account-service/Nimbus_aos/aos_all_in_one/ chmod +x startup.sh ./startup.sh For the DevOps story: cd account-service/Nimbus_aos/aos_pipline/aos_all_in_one/ chmod +x startup.sh ./startup.sh * Review and update the relevant fields in .env file (hidden file).   For machines in HPE network, don't forget to update the proxy fields.

Please note that:

- advantageonlineshopping.com is our production server and as such, is updated with version 1.1.4.

- advantageonlineshopping.com:8080 is our build server and as such, is updated nightly with the latest changes.
